Welcome to YuciferPay

YuciferPay is an automation service platform designed to make crypto payouts, payroll, and bulk/scheduled payments stress-free and simple.



Our Services

1. Bybit

This service is designed for Bybit merchants. We help you:

  • Retrieve orders automatically
  • Process payments
  • Confirm transactions
  • Send notifications to merchants

To get your Bybit API during registration, watch:
Bybit API Setup Guide

Important: Set your IP address to:
72.62.27.70
This is required for the service to function properly.



Bank Integration

You will need a bank API service for all the services we render. Currently, we support:

  • Paystack
  • Nomba Bank

You can:

  • Use one bank
  • Or switch between both anytime

Learn how to get your Paystack keys:
Paystack API Guide

Note: Make sure you are using LIVE mode.



2. Payroll

Our payroll system allows businesses to:

  • Pay employees automatically
  • Log payment records
  • Schedule recurring salary payments

Payments are tracked and updated monthly for continuous salary processing.



3. Payout

Designed for businesses handling bulk payments:

  • Send funds to multiple recipients
  • Schedule future payouts
  • Handle large-scale transactions easily

Unlike payroll, payout data is not stored after transactions.



Adding Users

You can add users:

  • Manually
  • Using a CSV file

This applies to both Payroll and Payout services.



Pricing

YuciferPay operates on a pay-per-transaction model.

  • No subscription fees
  • You will need to top up your topup balance, you are charged per payment made
